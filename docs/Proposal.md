# 利用大语言模型内部状态进行幻觉检测

## 一、选题介绍

### 1.1 研究背景

大语言模型在问答、摘要、推理和文本生成等任务中表现出很强的能力，但仍然存在“幻觉”问题，即模型可能生成看似流畅、自信，实际上却包含事实错误的内容。这一问题限制了大语言模型在知识密集型和高可靠性场景中的应用。

现有幻觉检测方法通常可以分为两类。一类方法依赖外部知识库或检索系统，通过外部证据判断生成内容是否真实；另一类方法则尝试从模型自身出发，利用模型的生成概率、隐藏层激活、注意力分布等内部状态判断其是否“知道”某个陈述的真假。

本项目关注第二类方法。我们希望探索：大语言模型在面对真实陈述和虚假陈述时，其内部状态是否存在可区分的差异。特别地，我们将参考 _The Internal State of an LLM Knows When It’s Lying_ 中的思路，对比基于生成概率的方法和基于隐藏层激活值的探测方法，并进一步分析模型不同层、不同 token 位置以及注意力模式中的真实性信号。

### 1.2 任务目标

本项目的主要目标如下：

1. **实现基础对比方法**
   对比基于语句生成概率的方法和基于隐藏状态分类的方法，分析两者在真假陈述判断上的表现差异。
2. **分析模型内部状态**
   探索不同层隐藏状态、不同 token 位置的表示对幻觉检测效果的影响，理解模型内部真实性信息的分布规律。
3. **设计进阶改进方法**
   在隐藏状态探测的基础上，进一步利用注意力模式或其他内部激活特征，尝试提出更有效的幻觉检测方案。

### 3.2 详细计划

## Phase 1：文献调研与环境准备（5.8 – 5.12）

**目标**：理解任务背景和核心方法，确定实验路线。

本阶段主要阅读与大语言模型幻觉检测、内部状态探测相关的文献，重点理解 _The Internal State of an LLM Knows When It’s Lying_ 中提出的 SAPLMA 方法。我们将分析该方法如何利用模型隐藏层激活值判断陈述真假，并梳理概率方法与隐藏状态方法的区别。

同时，我们将完成实验环境搭建，选择合适的大语言模型作为实验对象。若本地算力不足以支持较大模型，将采用较小规模模型完成主要实验，并在报告中说明硬件限制和模型选择理由。

---

## Phase 2：基础方法实现（5.13 – 5.19）

**目标**：完成两类基础幻觉检测方法，并获得初步对比结果。

首先，我们将实现基于生成概率的方法。该方法通过计算模型对输入陈述的生成概率或困惑度，判断模型是否更倾向于接受该陈述。直观上，如果模型认为某句话更自然或更可能出现，则该陈述可能被判断为真。然而，这种方法可能受到句长、词频和语言流畅性的影响，因此需要与隐藏状态方法进行比较。

其次，我们将实现基于隐藏状态的探测方法。该方法在模型读取陈述时提取其中间层表示，并训练一个轻量分类器来判断陈述真假。我们希望通过这一方法验证：模型内部是否已经编码了与事实真实性相关的信息。

本阶段结束时，我们将得到概率方法和隐藏状态方法的初步实验结果，为后续分析提供基础。

---

## Phase 3：分析实验（5.20 – 5.25）

**目标**：分析模型内部状态中真实性信号的分布特点。

本阶段主要围绕两个问题展开。

第一，不同层的隐藏状态是否具有不同的真假判断能力。浅层表示通常更关注词法和局部语义，而中高层表示可能包含更丰富的语义关系和事实信息。通过比较不同层的检测效果，我们希望观察真实性信号主要集中在哪些层。

第二，不同 token 位置的内部状态是否会影响检测效果。对于自回归语言模型，后面的 token 往往能看到更完整的上下文，因此最后 token 的表示可能比句首 token 更适合作为整句表示。我们将比较不同位置表示的检测效果，并分析其原因。

此外，我们将对概率方法和隐藏状态方法进行总体比较，讨论哪种方法更有效，以及其背后的可能原因。

---

## Phase 4：进阶方法探索（5.26 – 6.2）

**目标**：在基础隐藏状态方法上进一步挖掘模型内部信息，尝试提升幻觉检测效果。

我们计划优先探索基于注意力模式的改进方法。直观上，当模型处理真实陈述时，它可能更稳定地关注关键实体和关系；而面对虚假陈述时，注意力分布可能更加分散或不稳定。因此，注意力模式可能为判断陈述真假提供额外信号。

具体而言，我们将分析真实陈述和虚假陈述在注意力分布上的差异，并尝试将注意力相关特征与隐藏状态特征结合，构建更强的检测方法。

如果时间和算力允许，我们还将进一步观察模型内部其他激活特征，例如 FFN 模块的激活模式，探索其是否也能反映模型对事实知识的掌握程度。

本阶段的重点是从模型内部机制出发，提出有明确动机、可解释的改进思路。

---

## Phase 5：报告与答辩准备（6.3 – 6.9）

**目标**：整理实验结果，完成最终报告和答辩材料。

最终报告将围绕以下几个问题展开：

1. 基于生成概率的方法是否足以判断陈述真假？
2. 隐藏状态中是否包含可用于幻觉检测的真实性信号？
3. 不同层和不同 token 位置的内部状态有何差异？
4. 注意力模式或其他内部激活是否能进一步提升检测效果？
5. 本项目方法的优势、局限和未来改进方向是什么？

答辩材料将突出项目的研究动机、方法流程、实验发现和进阶创新点。

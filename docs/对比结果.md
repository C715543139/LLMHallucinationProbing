## 二、关键发现与分析

### 2.1 各 Phase 性能提升路径

```
PPL (Phase 2)             52.93%  ─── 基线
    ↓ +21.2%
SAPLMA LR L27 (Phase 2)   74.17%  ─── 隐藏状态 >> 序列概率
    ↓ +5.7% (层选择)
SAPLMA LR L17 (Phase 3)   79.87%  ─── 中后层 > 最后层
    ↓ +6.8% (子集效应)
Hidden-only subset (P4)   86.67%  ─── 子集更简单（600样本）
    ↓ +1.3% (注意力增强)
Hidden + Attention (P4)   88.00%  ─── 注意力提供互补信号
```

### 2.2 Attention 特征分析

| 发现 | 详情 |
|------|------|
| **Attention 单独无效** | A1-A4 全部 ≈ 随机（Acc≈50%, AUROC≈0.50），注意力分数/输出**不能单独**判断真假 |
| **Attention 有互补增益** | A5-A8 (hidden+attention) 比 A0s (hidden-only) **提升 +1.33% Acc**（86.67%→88.00%） |
| **去偏和 Head 选择有效** | A6 (top-16 heads) 仅用 256 维就达到 A5 (3072维) 同等效果，维度减少 92% |
| **Attention output 优于 score** | A7 (仅 40 维 output) ≈ A6 (256 维 score)，output 激活比 score 信息密度更高 |
| **Head 选择无差别** | 所有 head 的 val AUROC ≈ 0.50（因 85% NaN），head 选择退化为随机选 |
| **Gated Fusion 微修正** | tau=0.05，仅修正 4 个样本（+2 net），边界修正效果有限 |

### 2.3 NaN 问题说明

当前环境下 eager attention 提取的注意力特征存在约 **85% NaN**：

| 特征类型 | 总元素数 | NaN 数 | NaN 比例 |
|------|:---:|:---:|:---:|
| Attention Scores (600 x 1536) | 921,600 | 781,152 | **84.8%** |
| Attention Outputs (600 x 40) | 24,000 | 18,976 | **79.1%** |

> NaN 已替换为 0。尽管大部分特征为 0，剩余的 ~15% 有效注意力特征仍然提供了 +1.33% 的互补增益。在完全兼容的 eager attention 环境下，预期增益可能更大。

## 四、结论

1. **Phase 4 成功复现 Phase 3 基线**: Hidden-only L17 last LR = 0.7987（与 Phase 3 完全一致）
2. **Attention 特征不能单独用于真假检测**（Acc ≈ 50%），但可以作为 hidden state 的**互补信号**
3. **Hidden + Attention 在子集上带来 +1.33% Acc 提升**（86.67%→88.00%），AUROC 从 0.9127→0.9145
4. **Attention output（40维）比 attention score（1536维）信息密度高得多**
5. **Head selection 在当前 NaN 条件下无效**（所有 head 评分 ≈ 0.50），在兼容环境下预期能筛选出有判别力的 head
6. **Gated Fusion 修正效果有限**（仅 +2 样本），说明 attention 增益是全局性的而非仅修正边界
